## ğŸ› Troubleshooting

### Resume Not Parsing
- **Issue**: "Error parsing resume"
- **Solution**: 
  - Ensure PDF is not scanned image
  - Use text-based PDF
  - Try re-saving PDF from Word/Google Docs

### Fields Not Filling
- **Issue**: Some fields remain empty
- **Solution**:
  - Check if field labels match common patterns
  - Manually fill custom fields
  - Report issue for future improvement

### Multi-Page Not Working
- **Issue**: Second page doesn't autofill
- **Solution**:
  - Click autofill button again
  - Refresh page and retry
  - Check console for errors

### Platform Not Detected
- **Issue**: Shows "Unknown Platform"
- **Solution**:
  - Still works with generic autofill
  - May need manual review
  - Report platform for future support
